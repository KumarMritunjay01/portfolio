# MERN Portfolio

Modern portfolio website built using:

- React (Vite)
- Node.js
- Express.js
- MongoDB Atlas
- Tailwind CSS

## ✨ Features

✅ Responsive Design  
✅ Dark / Light Theme  
✅ Dynamic Projects  
✅ Contact Form  
✅ Modern UI  

## 🛠 Tech Stack

Frontend:
React + Tailwind

Backend:
Node.js + Express + MongoDB
